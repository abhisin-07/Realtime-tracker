# Realtime-tracker
The project is a real time tracker which allows the active users to see each others live location.
📍 Real Time Tracker

A Real-Time Location Tracking Web Application that allows users to share and view the live location of active users on an interactive map. The application continuously updates locations and displays them instantly for all connected users.

🚀 Features

📡 Real-Time Location Updates using WebSockets

🗺️ Interactive Map Integration for visualizing user locations

👥 Multiple User Tracking showing all active users on the map

⚡ Live Synchronization without page refresh

🔐 Efficient Backend Handling for real-time communication

🛠️ Tech Stack

Frontend

HTML

CSS

JavaScript

Map API (Leaflet / Google Maps)

Backend

Node.js

Express.js

Socket.IO

Other Tools

WebSockets for real-time communication

Geolocation API for fetching user coordinates

⚙️ Installation

Clone the repository

git clone https://github.com/yourusername/real-time-tracker.git

Navigate to the project folder

cd real-time-tracker

Install dependencies

npm install

Start the server

node server.js
▶️ Usage

Open the application in your browser.

Allow location access when prompted.

Your location will appear on the map.

You will also see locations of other active users in real time.
